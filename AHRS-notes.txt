4. Data Ingestion with GCP
    Install google cloud CLI
    run:
        gcloud --version
        pip install -e .

    In GCP > IAM & Admin > Service Accounts > Create Service Account >
    name: anime-hybrid-recommender-system (should be unique globally) >
    create and continue > roles: Storage Admin, Storage Object Viewer,
    Owner > continue & done

    In Service Accounts > ... menue of your Service Account > Manage Keys >
    Add Key > New Key > JSON > It will download a JSON file

    In bucket > ... menu of your bucket > edit access > Add Principal > 
    select your Service Account > role: Storage Object Admin, Storage Object Viewer

    run:
        set GOOGLE_APPLICATION_CREDENTIALS=paste path of key JSON file here

        pip install -e .
